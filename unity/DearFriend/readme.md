# Dear Friend — Unity Project

This folder contains the main Unity project for **Dear Friend**, a 3D interactive storytelling prototype created for the Care Workshop at HEAD Genève.

The experience follows Camille, a student helping Anna, an elderly resident, free space on her computer. Through file interactions, dialogue, camera movement, and environmental details, the player gradually discovers fragments of Anna’s life and memories.

The Unity project is used to build the interactive prototype, including the room scene, computer interface, character placement, animations, sound, and Yarn dialogue integration.

## Project Goals

This Unity project focuses on:

* creating an intimate 3D room environment
* testing the computer interface and file interactions
* integrating dialogue and story fragments
* testing the zoom-out camera mechanic
* placing Anna in the scene as a subtle but important presence
* building a playable prototype for iPad or desktop testing

## Prototype Features

The current prototype explores:

* a 3D room based on Anna’s living space
* an interactive computer interface
* clickable folders and files
* simple drag, click, or selection interactions
* camera framing and zoom-out moments
* Anna’s position in the room
* sound and dialogue moments
* Yarn-based narrative structure

## Development Notes

This project is still in development.

The prototype is mainly used to test:

* whether the player understands the computer interface
* how the zoom-out mechanic affects the feeling of intimacy or distance
* how visible Anna should be during the experience
* how the room layout supports the story
* how the pacing of the interaction affects emotional engagement
